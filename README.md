# Distribution of Smoking Behaviors in the U.S.

This Shiny app explores the distribution of smoking behaviors across U.S. states using BRFSS data from the CDC.

## Features
- Interactive histogram visualization
- Adjustable bin sizes
- Custom color selection
- Summary statistics
- Mean reference line

## Data Source
Behavioral Risk Factor Surveillance System (BRFSS), CDC.
