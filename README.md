# Distribution of Smoking Behaviors in the U.S.

## Research Question
What is the distribution of smoking behaviors across the U.S. population?

## Overview
This Shiny app explores the distribution of smoking behaviors across U.S. states 
using data from the Behavioral Risk Factor Surveillance System (BRFSS). The app 
provides interactive histograms and summary statistics that allow users to 
explore variability in smoking prevalence across multiple smoking categories.

## Features
- Interactive histogram visualization
- Adjustable histogram bin sizes
- Custom color selection
- Summary statistics output
- Mean reference line for interpretation

## Methods
The app uses descriptive statistical methods and histogram visualizations to 
analyze smoking prevalence data across U.S. states. Users can interactively 
select smoking categories and customize visualization settings.

## Data Source
Behavioral Risk Factor Surveillance System (BRFSS), Centers for Disease Control 
and Prevention (CDC).
